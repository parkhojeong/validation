spring boot auto configure
- register LocalValidatorFactoryBean to global Validator
- @Valid(or @Validated) field annotation
- validation error store in BindingResult(FieldError, ObjectError)

validation flow 
1. bind value to field(@ModelAttribute)
    - success 
    - failure -> add typeMismatch FieldError
2. apply Validator

* apply Bean Validation to field binding success

annotation
- @Valid - java bean validation
- @Validated - spring bean validation. group validation
