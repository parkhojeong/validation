[BindingResult]
- store validation error
- if BindingResult declared, handle @ModelAttribute data binding error
  - if BindingResult isn't declared, controller method not called
    - 400 error occurred
  - if BindingResult is declared, controller method called 
    - BindingResult including FieldError

[apply Validation error to BindingResult]
- method parameter(@ModelAttribute, BindingResult) 
    - spring will bind error to BindingResult
- program call BindingResult.addError()
- Validator

* caution
- parameter order important (@ModelAttribute, BindingResult)
- Model auto has BindingResult
